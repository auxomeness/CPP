# C++ Practice and Exercises Collection

This repository contains a collection of C++ programs developed for practice, coursework, and problem-solving exercises. The files cover a wide range of fundamental programming concepts including control structures, arrays, strings, recursion, sorting algorithms, file handling, and data structures.

---

## Overview

The purpose of this repository is to:

* Strengthen core C++ programming skills
* Practice problem-solving techniques
* Explore different algorithmic approaches
* Serve as a reference for basic to intermediate C++ concepts

---

## Requirements

* C++ compiler (g++ recommended)
* Any OS (Windows, Linux, macOS)
* Optional: IDE such as VS Code, Code::Blocks, or CLion

---

## How to Compile

Use the following command to compile any file:

```bash
g++ filename.cpp -o output
```

Example:

```bash
g++ bubbleSort.cpp -o bubbleSort
```

---

## How to Run

```bash
./output
```

On Windows:

```bash
output.exe
```

---

## Notes

* Each `.cpp` file is a standalone program with its own `main()` function
* Files are independent and not intended to be compiled together
* Some filenames include spaces—use quotes when compiling:

```bash
g++ "DIGIT SUM.cpp" -o digitsum
```
